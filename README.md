nestingBox
==========

Benötigte Teile:

Technische Ausstattung:
- Raspberry Pi
- Raspberry Pi NoIR Kamera Modul
- USB WLAN Adapter oder LAN
- Lochrasterplatine
- Kabel
- 4 IR-LED-Dioden + Widerstände
- Makrolinse (+4) einer Digitalkamera (Close-Up Lense)

Nistkasten:
- Holz
- Dachpappe
- 2 Scharniere
- Rundstab (ca. 6cm)
